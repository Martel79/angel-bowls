Men's Selectors' Portal

Angel Indoor Bowls Club

This portal stores all match and squad data in Google Firebase — a free cloud database. Once set up, any device with this file can access and edit the same shared data. Follow the steps below (takes about 5 minutes).

Step-by-step setup

1Create a free Firebase project

Go to console.firebase.google.com and sign in with your Google account.

Click Add project, name it angel-bowls, disable Google Analytics (not needed), then click Create project.

2Create a Firestore database

In the left sidebar click Build → Firestore Database, then Create database.

Choose Start in production mode, pick europe-west2 (London) as your region, and click Enable.

3Set security rules (allow club access)

In Firestore, click the Rules tab. Replace the rules with:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}

Click Publish. (This allows anyone with the config to access the data — suitable for a private club file. You can tighten this later.)

4Register a Web App & get your config

In the Firebase console, click the ⚙️ gear icon → Project settings.

Scroll to Your apps, click the </> (Web) icon, name it selectors-portal, and click Register app.

You'll see a firebaseConfig object with several values. Copy each value into the fields below.

5Paste your config values below
API Key
Auth Domain
Project ID
Storage Bucket
Messaging Sender ID
App ID

Your config is saved only in this browser's local storage — it is never sent anywhere else. Share this .html file with other selectors; they each enter the same config once on their own device.
